ABSTRACT

Microstrip antenna refers to an antenna fabricated which uses microstrip techniques on a printed circuit board. It is one type of internal antenna which is used mainly in devices and microwave frequencies. The patch antenna has been rapidly used in various fields like space technology, aircrafts, missiles, mobile communication, GPS system, and broadcasting. A microstrip patch antenna is made out of a trace of copper or some other metal of any geometry on one side of a standard printed circuit board (PCB) substrate with opposite side grounded. This antenna can be fed utilizing coaxial, stripline, opening coupling or vicinity coupling strategies. This Antenna is getting popularity over the decades because it has some very well-known advantages. Such as- light weight, low profile, easy and low cost fabrication, planar but also conformal to non-planar geometrics, mechanical robustness, easy integration of components, easy association in arrays and versatility in terms of electromagnetic characteristics.

This work was started with designing Microstrip Patch antenna with different shaped. This work includes circular, F letter shaped, triangular, square and hexagonal shape antenna. And the most important part is all these antennas are designed with same area sand same frequency range. Among these shapes, Circular shape performs well comparing with others. The parameters like- Return loss, bandwidth, gain etc. which are calculated throughout the work showed a good result for circular shape. The designing and simulation of the antenna is carried out using CST microwave studio and fabrication is done in our laboratory.
